# ToDo-List

ToDo list is a simple Web application to save your daily task.

## Features

* Add new task
* Edit task
* Mark task as completed or uncompleted
* Delete task

## Technologies used:

* HTML
* CSS
* React
* [Ruby on Rails](https://github.com/Jcheahh/todo-backend)
